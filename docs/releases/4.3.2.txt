Version 4.3.2
=============

Changes
-------


File metadata
^^^^^^^^^^^^^

Use the correct icon for the document type file metadata setup link.

^^^^^^

The response structure of the search model API view was fixed. The search
fields for each search model are now correctly displayed.


Views
^^^^^

Fixed the last hard coded instance of the key used to determined the list
mode.

- Merge bugfix version 4.2.11 and 4.2.12.
- Update translation files.
